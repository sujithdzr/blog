Codemancers at RubyConf India 2014

Hemant, Director | Rubyconf, Startup, India

Codemancers is one of the silver sponsors for RubyConf India which is happening on the 22nd and 23rd (Sat & Sun) of this month in Goa. Click here for the complete event schedule.

On Saturday(22nd), Kashyap, from our team will be holding a session on 'Building your own Ruby profiler’.

Some of the internal changes and tools introduced in Ruby 2.1 make it easy to develop high-level profilers. This talk will not only address the changes and the instrumentation provided in the latest version, but will also take you through the details to build a basic profiler using the tools.

Deepak’s session on ‘Build your own code metrics tool for Ruby’s is on the 23rd (Sun).

Generating metrics is probably easy. But how you visualize them and how you use them is fairly complex. In this session, Deepak will talk about the kind of metrics which we might be interested in, how to generate those metrics ourselves and talk about the internal workings of some of the existing tools.

Apart from Deepak and Kashyap, you will find Yuva, Emil, Girish and Sujith from Codemancers. We should be easy to spot in our Codemancers Tees.

See you in Goa!!
